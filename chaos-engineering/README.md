# SRE Philosophy Tool

A comprehensive toolkit for implementing Site Reliability Engineering (SRE) practices.

## Repository Structure

This repository contains multiple modules for different aspects of SRE:

- **[synthetic-monitoring](./synthetic-monitoring/README.md)** - Chaos engineering with VMware Mangle, SLI/SLO/SLA monitoring with Wavefront, and user metrics measurement
- **[chaos-engineering](./chaos-engineering/README.md)** - Chaos engineering workflows and documentation
- **[chaos-engineering/SECURITY-SCANS.md](./chaos-engineering/SECURITY-SCANS.md)** - Security scanning tools and workflows documentation

## Quick Links

## Overview

This toolkit helps organizations implement SRE principles through:
- Chaos engineering and fault injection
- Service level monitoring (SLI/SLO/SLA)
- Security scanning and vulnerability management
- User metrics measurement and analysis
